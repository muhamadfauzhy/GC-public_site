# Catatan Week 2

> Tuliskan apapun yang kalian pelajari pada phase 2 week-2 di file ini.
>
> "Pemahaman yang baik berasal dari keinginan untuk terus belajar, dan catatan adalah langkah pertama menuju pengetahuan yang dalam."

Pada phase 2 week 2 ini, saya mempelajari beberapa hal penting terkait pengembangan aplikasi berbasis React dan integrasi dengan backend API.

Hal utama yang saya pelajari adalah bagaimana membuat form yang bersifat controlled component di React menggunakan useState. Saya juga memahami pentingnya menjaga konsistensi antara value dan onChange agar tidak terjadi error seperti controlled dan uncontrolled component.

Selain itu, saya belajar:

Cara menggunakan axios untuk melakukan request ke backend (GET, POST)
Mengimplementasikan fitur CRUD (Create, Read, Update, Delete)
Menggunakan useEffect untuk mengambil data dari API
Mengelola props dan state untuk mengirim data antar komponen
Menangani error dari backend dan menampilkan notifikasi menggunakan Toastify
Menggunakan React Router (useNavigate, useParams) untuk navigasi dan mengambil parameter dari URL

Saya juga memahami pentingnya:

Konsistensi penamaan field (misalnya CategoryId vs categoryId)
Validasi data sebelum dikirim ke backend
Debugging menggunakan console.log untuk melacak error

Dari pembelajaran ini, saya jadi lebih memahami bagaimana frontend dan backend saling terhubung, serta bagaimana membuat aplikasi yang lebih interaktif dan dinamis.
